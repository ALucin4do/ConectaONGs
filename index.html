CONECTAONGS

Este agregador de ONGs foi desenvolvido com o intuito de consolidar e aprimorar minhas habilidades em HTML e CSS. A aplicação utiliza conceitos como Flexbox para criar layouts flexíveis e adaptáveis a diferentes dispositivos. Através deste projeto, busquei praticar o HTML semântico para garantir uma estrutura clara e organizada do conteúdos como as informações das ONGs, e a construção de uma interface intuitiva e agradável ao usuário.
